Как использовать
------------------
1. Импорт в приложении: режим «одна ТТК» для файлов synth__1card__*, «несколько ТТК» для synth__multi__*.
2. После проверки — «Создать все», чтобы записалось обучение по подписи заголовка.
3. Повторяйте для разных файлов — расширяется покрытие форматов.
