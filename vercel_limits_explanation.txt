ОГРАНИЧЕНИЯ VERCEL АККАУНТА

Vercel имеет разные тарифные планы с ограничениями:

HOBBY (бесплатный):
- 100 GB bandwidth в месяц
- 3000 функции в месяц
- 100 deployments в месяц
- Ограниченное использование Deploy Hooks
- Нет поддержки Enterprise функций

PRO:
- 3000 GB bandwidth
- Неограниченные функции
- 3000 deployments
- Продвинутые Deploy Hooks
- Аналитика, monitoring

ENTERPRISE:
- Неограниченные ресурсы
- Кастомные домены
- Advanced security
- SLA гарантии

ПРОБЛЕМА:
Deploy Hook может не работать на Hobby плане из-за ограничений API вызовов.

РЕШЕНИЕ:
- Обновить план до Pro
- Или использовать ручной redeploy через Dashboard
- Или перейти на GitHub Actions для автоматизации
