# Возможность интеграции нейросети в Restodocks

## Кратко

**Да, это возможно.** Нейросеть можно использовать для «умной» обработки данных: понимание, распределение, категоризация.

## Что можно делать

1. **Разбор и категоризация продуктов**
   - Загрузка текста/Excel → модель определяет тип (овощ, мясо, молочное и т.п.) и подбирает категорию
   - Нормализация названий (синонимы, опечатки)

2. **Распределение данных**
   - Автоматическое отнесение строк к нужным полям (продукт, категория, единица измерения)
   - Работа с «сырыми» списками без фиксированного формата

3. **Дополнительные сценарии**
   - Подбор категории для нового продукта
   - Подсказки при создании ТТК

## Как это обычно реализуется

- **Backend/Edge Function** (Supabase Edge Functions, Cloud Functions и т.п.)
- Вызов API (OpenAI, Anthropic, локальные модели)
- Обработка на сервере, клиент отправляет данные и получает структурированный результат

## Важно

- Нужен API-ключ и оплата запросов
- Важно обеспечить приватность (не передавать лишние данные)
- Для начала хватает простого сценария: «отправить строки → получить категории и нормализованные названия»

## Следующий шаг

Определить первый конкретный сценарий (например, «умная» категоризация при загрузке текстового списка) и добавить его в приложение через Edge Function.
