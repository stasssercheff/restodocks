# Сохранение графика и поставщиков в облаке

1. [Supabase Dashboard](https://supabase.com/dashboard) → проект → **SQL Editor** → **New query**
2. Откройте файл **APPLY_SCHEDULE_ORDER_LISTS_MIGRATION.sql** — скопируйте **всё** содержимое (там только SQL)
3. Вставьте в Supabase и нажмите **Run**

Важно: копировать нужно именно .sql файл. В нём нет заголовков и markdown — Supabase не выдаст ошибку.
