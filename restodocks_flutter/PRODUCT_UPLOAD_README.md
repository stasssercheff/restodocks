# Как добавлять продукты в номенклатуру

## Быстрый старт

1. **Откройте приложение** и войдите в свой аккаунт
2. **На главном экране** нажмите на плитку "Загрузить продукты"
3. **Выберите способ загрузки:**
   - **Из файла**: Excel (.xlsx, .xls), Текст (.txt), RTF (.rtf)
   - **Вставить текст**: Скопируйте список из любой таблицы

## Формат данных

### Простой формат (рекомендуется)
```
Название [Tab] Цена
```

Примеры:
```
Авокадо	₫99,000
Базилик	₫267,000
Молоко	₫38,000
Картофель	₫25,000
```

### Альтернативные разделители
- **Табуляция** (Tab) - идеально
- **Несколько пробелов** - тоже работает
- **Запятые в ценах** - поддерживаются (₫99,000)

## Где взять список продуктов?

### Вариант 1: Из Excel
1. Создайте таблицу с колонками: Название | Цена
2. Сохраните как .xlsx или .xls
3. Загрузите в приложение

### Вариант 2: Из текста
1. Скопируйте список из любого документа
2. Вставьте в поле "Вставить текст"
3. Нажмите "Добавить"

### Вариант 3: Из другого источника
- Скопируйте из PDF (через предварительный просмотр)
- Скопируйте из веб-страницы
- Используйте любой текстовый редактор

## Что делает ИИ?

При загрузке ИИ автоматически:
- ✅ **Исправляет опечатки** в названиях
- ✅ **Определяет категории** продуктов (овощи, мясо, специи...)
- ✅ **Предлагает единицы измерения** (кг, г, шт, л...)
- ✅ **Проверяет цены** и предлагает корректные
- ✅ **Добавляет калории** если известны

## Результат

После загрузки продукты:
- Добавляются в **общую базу данных** (доступны всем)
- Добавляются в **вашу номенклатуру** заведения
- Становятся доступны для создания **ТТК** и **меню**

## Советы

- **Начинайте с малого**: сначала загрузите 5-10 продуктов для проверки
- **Проверяйте результат**: после загрузки посмотрите в разделе "Номенклатура"
- **Используйте понятные названия**: они будут отображаться в меню
- **Добавляйте цены**: это поможет при расчете стоимости блюд

## Устранение проблем

### Продукты не загрузились
- Проверьте формат данных (название + Tab + цена)
- Убедитесь, что файл не поврежден
- Попробуйте вставить текст вместо файла

### Неправильные цены
- Проверьте разделители (запятые, пробелы)
- Убедитесь, что цены без текста (только числа и валюта)

### Ошибки при загрузке
- Попробуйте перезагрузить приложение
- Проверьте подключение к интернету
- Если проблема persists, обратитесь к администратору
