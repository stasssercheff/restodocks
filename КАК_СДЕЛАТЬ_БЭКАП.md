# Как сделать полный бэкап Restodocks

## Простой способ (1 клик)

1. Найди в папке проекта файл **`Backup_Restodocks.command`**
2. Двойной клик по нему
3. Подожди 1–3 минуты
4. Готово. Появится файл `backup_20260225_XXXXXX_COMPLETE.tar.gz`

---

## Если нужно, чтобы в бэкап попала база данных

Сейчас база может не сохраняться, если не установлен pg_dump.

**Установи Postgres.app:**
1. Открой https://postgresapp.com
2. Скачай и установи (перетащи в Programs)
3. Запусти Postgres.app → нажми «Initialize»
4. Готово

После этого снова запусти бэкап (двойной клик по `Backup_Restodocks.command`).

В конце будет написано «База данных PostgreSQL ✓» — значит всё попалo в архив.
