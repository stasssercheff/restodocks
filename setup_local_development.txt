НАСТРОЙКА ЛОКАЛЬНОЙ РАЗРАБОТКИ

1. ПРОВЕРКА КОНФИГУРАЦИИ:
   - config.json: содержит production ключи
   - .env: содержит development ключи
   - Нужно выбрать правильный набор ключей

2. ЗАПУСК ЛОКАЛЬНО:
   cd restodocks_flutter
   flutter run -d chrome

3. ЛОКАЛЬНЫЙ СЕРВЕР:
   flutter run -d web-server --web-port=3000

4. ДОСТУП К СЕРВИСАМ:
   - Supabase: работает с любыми ключами
   - GitHub: локально не нужен
   - Vercel: только для деплоя

5. ТЕСТИРОВАНИЕ:
   - Открыть http://localhost:3000
   - Проверить все функции
   - Загрузка продуктов, инвентаризация

ЛОКАЛЬНАЯ ВЕРСИЯ ДОЛЖНА РАБОТАТЬ С SUPABASE!
